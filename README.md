## ✨ What is it?

A lightweight, zero-dependency utility that capitalizes the first letter of each word in a string. Perfect for formatting names, titles, headings, or any text that needs proper capitalization - all with full TypeScript support.

> ✅ **Production Ready:** Battle-tested with 100% test coverage and used in production applications.

## 🎯 Key Features

- 📝 **Smart Capitalization:** Capitalizes first letter of each word while preserving existing case
- 🎨 **Flexible Options:** Configure lowercase conversion and exception words
- 🔤 **Unicode Support:** Works with international characters and emojis
- 📦 **Zero Dependencies:** Lightweight and fast - no npm packages required
- 🔷 **TypeScript Ready:** Full type definitions included
- 🧪 **Well Tested:** Comprehensive test suite with 100% coverage
