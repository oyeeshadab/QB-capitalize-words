# capitalize-words

Capitalize the first letter of each word in a string with full TypeScript support.

## Features

- ✅ Full TypeScript support with type definitions
- ✅ Zero dependencies
- ✅ Unicode support
- ✅ Configurable options
- ✅ Exception words
- ✅ Comprehensive test coverage
- ✅ Tree-shakeable

## Installation

```bash
npm install @quasarbytes/capitalize-words
```
